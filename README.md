# 关系动力学全息

> 世界最顶级的两性关系感知训练系统

## 📖 简介

这是一个基于「对比驱动学习」的本地化训练系统，帮助你从第零阶（默认沉默）逐步进阶到第八阶（被爱）。

### 核心哲学

```
信息 → 情绪 → 感受 → 看见 → 欣赏 → 理解 → 爱 → 被爱
```

**信息让两个人相遇，情绪让两个人靠近，感受让两个人连接，看见让两个人惊喜，欣赏让两个人发光，理解让两个人深刻，爱让两个人扎根，被爱让两个人完整。**

## 🚀 快速开始

### 方式一：自动启动

```bash
chmod +x start.sh
./start.sh
```

如果 3000/8000 端口上残留的是本项目旧进程，启动脚本会自动清理并重新启动；如果端口被其他程序占用，脚本会提示你先关闭该程序。

### 方式二：手动启动

**后端：**
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -c "from backend.database.seed import seed_all; seed_all()"
uvicorn backend.main:app --host 127.0.0.1 --port 8000
```

**前端：**
```bash
cd frontend
npm install
VITE_API_PROXY_TARGET=http://127.0.0.1:8000 npm run dev -- --host 127.0.0.1 --port 3000
```

### 访问

- 前端界面：http://localhost:3000
- 后端API：http://localhost:8000
- API文档：http://localhost:8000/docs

## 🎯 核心功能

- 今日训练统计

- 游戏库：破冰游戏
- 急转弯：浪漫急转弯

### 4. 八阶路径 (Path)
- 从默认沉默到无话不谈
- 每个阶段的定义、目标、工具、升级条件

### 5. 复盘日记 (Journal)
- 每日5W2H复盘
- 情绪记录
- 改进追踪

### 6. 我的档案 (Profile)
- 依恋类型
- 能力雷达图
- 学习统计

## 🧠 第一性原理

| 原理 | 表述 | 系统实现 |
|------|------|----------|
| 感知先于认知 | 没有细微的感知，所有高阶认知都是空中楼阁 | 训练从微观线索识别开始 |
| 对比驱动学习 | 人类只能通过"差异"学习 | 每个环节嵌入原回应 vs 理想回应对比 |
| 情绪是唯一货币 | 关系中没有对错，只有情绪是否被看见、被接住 | 所有样本的核心是情绪流动轨迹 |
| 二八定律 | 20%的感知维度决定80%的关系质量 | 优先训练情绪强度标尺、依恋信号、边界试探 |

## 📊 八阶路径

| 阶段 | 名称 | 目标 |
|------|------|------|
| 0 | 沉默 | 启动侦探模式 |
| 1 | 信息 | 交换基本信息 |
| 2 | 情绪 | 识别情绪标签 |
| 3 | 感受 | 让她感到"你和我在一起" |
| 4 | 看见 | 照出她未意识到的细微状态 |
| 5 | 欣赏 | 用具体真诚的光照亮她 |
| 6 | 理解 | 建立她的内在模型 |
| 7 | 爱 | 稳定的行动 |
| 8 | 被爱 | 安心接收并回馈爱的循环 |

## 📦 数据模型

### 情绪谱系
- 7大谱系 × 10级强度 = 70个基本情绪词
- 20+种混合情绪（酸楚、纠结、心酸等）

### 互动样本
- 场景分类：初识/暧昧/热恋/冲突/平淡/修复
- 难度等级：初级/中级/高级
- 完整标注：情绪标签、强度、需求、依恋信号、边界测试
- 回应对比：错误回应 vs 理想回应（柔和/张力/幽默三版）
