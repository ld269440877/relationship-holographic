# 对比引擎使用指南

## 概述

对比引擎是微观关系动力学全息系统的核心组件，负责：
1. 生成对比学习数据
2. 计算差异度
3. 生成可视化报告
4. 追踪进步轨迹

## 核心功能

### 1. 对比类型

| 类型 | 说明 | 应用场景 |
|------|------|----------|
| 情绪标尺对比 | 你的判断 vs 专家判断 | 情绪识别训练 |
| 回应风格对比 | 你的回应 vs 三种风格 | 回应优化训练 |
| 时间轴对比 | 今天 vs 昨天 vs 上周 vs 上月 | 进步追踪 |
| 横向对比 | 你 vs 平均水平 vs 优秀用户 | 定位自身水平 |

### 2. 使用示例

```javascript
import { ContrastEngine } from '@/compare/engine';

// 创建对比引擎实例
const engine = new ContrastEngine();

// 生成对比数据
const contrast = await engine.generate({
  type: 'response',
  userResponse: '那你早点休息',
  idealResponse: '听起来今天很耗你，想聊聊吗？',
  scene: {
    her_words: '今天好累啊',
    her_emotion: '疲惫',
    context: '工作了一天'
  }
});

// 渲染对比视图
const view = engine.render(contrast);
```

### 3. 差异分析

引擎会自动分析以下差异点：
- 情绪词使用
- 需求回应
- 表达温度
- 问句类型
- 留白程度

## API 参考

### ContrastEngine

```javascript
class ContrastEngine {
  // 生成对比数据
  async generate(options)
  
  // 渲染对比视图
  render(contrastData)
  
  // 计算差异度
  calculateDiff(response1, response2)
  
  // 生成进步报告
  async generateProgressReport(userId, period)
}
```

### 方法详解

#### generate(options)

生成对比数据

**参数**:
- `type`: 对比类型 ('response' | 'emotion' | 'scene')
- `userResponse`: 用户的回应
- `idealResponse`: 理想回应
- `scene`: 场景信息（可选）

**返回**: 
```javascript
{
  highlights: [...], // 高亮差异
  analysis: {...},    // 差异分析
  score: number,      // 匹配度分数
  suggestions: [...]  // 改进建议
}
```

#### render(contrastData)

渲染对比视图

**返回**:
```javascript
{
  html: string,       // HTML字符串
  css: string,         // 样式字符串
  charts: [...]       // 图表数据
}
```

## 数据结构

### 差异节点

```javascript
{
  type: 'emotion_word' | 'need_response' | 'temperature' | 'question_type' | 'whitespace',
  user: string,        // 用户使用的
  ideal: string,       // 理想使用的
  diffLevel: 'missing' | 'different' | 'extra',
  suggestion: string  // 改进建议
}
```

### 进步报告

```javascript
{
  period: 'day' | 'week' | 'month',
  metrics: {
    emotionAccuracy: { before: number, after: number, change: number },
    responseQuality: { before: number, after: number, change: number },
    speed: { before: number, after: number, change: number }
  },
  highlights: [...],
  nextGoals: [...]
}
```

## 使用场景

### 场景1：训练后对比

用户完成训练后，系统自动生成对比报告：
- 原回应 vs 理想回应
- 差异高亮
- 改进建议

### 场景2：每日复盘

用户查看每日进步：
- 时间轴对比
- 进步曲线图
- 薄弱点分析

### 场景3：横向对比

用户定位自身水平：
- vs 平均用户
- vs 优秀用户
- 差距分析

## 最佳实践

1. **每次训练后生成对比数据**
2. **每周查看进步报告**
3. **关注差异趋势而非绝对分数**
4. **重点改进高频出错维度**
