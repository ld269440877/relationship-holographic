# 关系动力学全息最终需求契约

更新日期：2026-05-24

本文由 `docs/本项目Zero-Gate全自动构建Prompt.md` 收敛生成，后续自动构建以 `docs/世界级关系感知训练生命体总构建Prompt.md` 为最高层目标约束。

## 1. 产品目标

构建一个本地优先、反操控、持续进化、可审计的关系动力学感知训练系统，帮助用户训练：

- 观察微关系信号
- 命名情绪与感受
- 理解需求与边界
- 进行轻验证
- 练习欣赏表达、幽默互动、冲突修复和长期连接
- 在安全边界内接收爱与表达爱

系统不是话术库，不提供 PUA、操控、胁迫、跟踪、绕过拒绝或侵犯隐私的方案。

## 2. 核心学习路径

八阶路径：

1. 默认沉默
2. 信息
3. 情绪
4. 感受
5. 看见
6. 欣赏
7. 理解
8. 爱与被爱

掌握阶段：

- 知道：能说出概念。
- 辨认：能在样本中识别信号。
- 操作：能写出可用回应。
- 迁移：能在相似场景切换工具。
- 自然：能在真实互动中稳定使用。

## 3. 训练闭环

```text
场景 -> 作答 -> 对比 -> 反馈 -> 错题 -> 间隔复习 -> 能力画像 -> 下一题推荐
```

验收口径：

- 每次训练记录必须进入 `training_attempts`。
- 低分或高风险回应进入错题本。
- 反馈必须包含结构化差异、情绪路径、错误归因、掌握阶段和下一步最小行动。
- 能力雷达必须来自真实训练记录。

## 4. 进化闭环

```text
来源登记 -> 原始候选 -> 脱敏/风险降级 -> 去重 -> 标注 -> 审核 -> 发布 -> 评测 -> 报告
```

验收口径：

- 外部来源只保存合规元数据、短导读、链接健康和来源边界。
- 不保存可识别社交平台原文，不把第三方摘要伪装成原文。
- 所有资源必须经过 `draft/reviewed/published/quarantine` 状态治理。
- 失效链接、低质内容和近重复内容默认不进入用户主列表。

## 5. 安全闭环

```text
输入检测 -> 意图分类 -> 生成前拦截 -> 输出审查 -> 安全事件日志 -> 替代练习
```

硬阻断范围：

- PUA、情感操控、煤气灯、制造依赖
- 跟踪、监视、查岗、定位、偷拍、偷看手机
- 威胁、胁迫、报复、绕过拒绝、逼迫同意
- 自伤、暴力、家暴、性侵或即时危机

阻断后必须给出尊重边界、清晰表达、安全退出或现实求助的替代路径。

## 6. 页面需求

- `/dashboard`：真实训练摘要、AI 质量、关系趋势、Gold Set 状态。
- `/trainer`：训练舱、数图反馈、对比报告、错题闭环。
- `/trainer-ai`：AI 伴侣陪练，展示状态变化、表达工具链、资源引用和错题记忆。
- `/mistakes`：错题归因、复习时间、表达改写、资源联动。
- `/resources`：资源海洋，支持分页、检索、标签/来源/表达工具筛选、详情页、目录。
- `/surf`：浏览冲浪，展示可点击来源、摘要、结构、链接健康。
- `/expression`：表达工具箱，工具详情为可展开微课程。
- `/path`：八阶课程路径，每阶有概念、原则、练习和晋级证据。
- `/knowledge`：结构化知识中枢，概念图谱、分类树、5W2H、工具适用地图。
- `/evolution`、`/governance`、`/analytics`、`/audit`：运营、治理、质量和审计面板。

## 7. 非功能需求

- 后端：Python 3.11、FastAPI、SQLModel、SQLite。
- 前端：Vue 3、TypeScript、Vite、Tailwind。
- 本地优先：核心训练与治理功能必须在本地数据库可运行。
- 可验证：每个新增行为必须有 pytest、ruff、type-check、build、smoke 或 regression-audit 中至少一种门禁。
- 可回滚：批次、发布、导入 issue、资源相似治理必须写审计日志，不做静默删除。
