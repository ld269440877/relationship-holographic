## 执行原则

1. 不再追求“数据库数字巨大”本身，优先提高每条内容的具体性、可练性、边界清晰度、情绪流动和来源可信度。
2. 不把第三方网页摘要伪装成原文；外部来源只保存合法元数据、短导读、来源边界和用户跳转链接。
3. 所有自动扩充都要进入 reviewed/published/quarantine 治理流，不允许绕过发布门禁。
4. 先修 P0 可见体验和质量门禁，再做 P1 扩容和 P2 长期自动化。
5. 每项任务必须有验证命令或浏览器 smoke，不做无法验收的“全面优化”。
