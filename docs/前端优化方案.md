## 一、问题总览

### 1.1 内容层问题

| 位置 | 问题 | 严重度 |
|------|------|--------|
| Dashboard 统计 | 所有数字 mock，连胜/准确率/已学样本全是假数据 | 🔴 高 |
| Dashboard 八阶进度 | radarData 来自 store但数据是随机的，不是真实计算 | 🔴 高 |
| Dashboard 推荐训练 | 硬编码的3条固定文案 | 🟡 中 |
| Path 页面 | 所有进度/完成状态硬编码 | 🔴 高 |
| Resources | 无详情页，只能看卡片摘要 | 🟡 中 |
| Journal | 复盘问题太浅，缺乏引导性 | 🟡 中 |
| Profile | 依恋测试入口存在但无测评 | 🟡 中 |
| Trainer | AI伴侣模拟是空概念，无真实对话体验 | 🟡 中 |

### 1.2 结构层问题

| 问题 | 描述 |
|------|------|
| 缺少新用户引导 | 没有 onboarding 流程，用户进来直接面对 Dashboard |
| 缺少每日微训练节拍 | 没有"每日任务包"的概念，用户不知道每天该做什么 |
| 缺少错题本完整页 | Dashboard 只有3条摘要，需要独立页面 |
| 缺少场景样本详情 | 资源卡片点击无展开，内容深度看不到 |
| 缺少情感日报/周报 | 没有周期性数据回顾 |
| 缺少 AI 伴侣独立页面 | AI 模拟嵌套在 Trainer 模式下，体验割裂 |

```vue
      <p class="text-sm text-gray-500 mt-3">{{ selectedBot?.description }}</p>
    </div>

    <!-- 对话区域 -->
    <div ref="chatContainer" class="flex-1 overflow-y-auto p-6 space-y-4">
      <!-- 欢迎消息 -->
      <div v-if="messages.length === 0" class="text-center py-12 text-gray-400">
        <p class="text-5xl mb-4">{{ selectedBot?.avatar }}</p>
        <p>{{ selectedBot?.welcome }}</p>
        <p class="text-sm mt-2">发送消息开始对话</p>
      </div>

      <!-- 消息列表 -->
      <div v-for="(msg, i) in messages" :key="i"
           class="flex" :class="msg.from === 'user' ? 'justify-end' : 'justify-start'">
        <div class="max-w-xl rounded-2xl px-4 py-3"
             :class="msg.from === 'user'
               ? 'bg-blue-500 text-white rounded-br-sm'
               : 'bg-gray-100 dark:bg-gray-700 rounded-bl-sm'">
          <p>{{ msg.text }}</p>
          <p class="text-xs opacity-60 mt-1">{{ msg.time }}</p>
        </div>
      </div>

      <!-- 加载中 -->
      <div v-if="loading" class="flex justify-start">
        <div class="bg-gray-100 dark:bg-gray-700 rounded-2xl px-4 py-3">
          <p class="text-gray-400 animate-pulse">AI 正在思考...</p>
        </div>
      </div>
    </div>

    <!-- 评分提示（对话结束后） -->
    <div v-if="showScore" class="p-4 bg-gradient-to-r from-blue-500 to-purple-500 text-white">
      <p class="text-center mb-4">本次对话评分</p>
      <div class="flex justify-center gap-8 text-center">
        <div>
          <p class="text-3xl font-bold">{{ score.emotionAccuracy }}</p>
          <p class="text-xs opacity-80">情绪感知</p>
        </div>
        <div>
          <p class="text-3xl font-bold">{{ score.responseQuality }}</p>
          <p class="text-xs opacity-80">回应质量</p>
        </div>
        <div>
          <p class="text-3xl font-bold">{{ score.empathy }}</p>
          <p class="text-xs opacity-80">共情表达</p>
        </div>
      </div>
    </div>

    <!-- 输入区域 -->
    <div class="p-4 border-t dark:border-gray-700">
      <div class="flex gap-3">
        <input v-model="inputText" @keydown.enter="sendMessage"
               class="flex-1 input-mac" placeholder="输入你的回复..."
               :disabled="loading" />
        <button @click="sendMessage" class="btn-primary px-6" :disabled="!inputText.trim() || loading">
          发送
        </button>
      </div>
      <div class="flex gap-2 mt-2">
        <button v-for="hint in quickHints" :key="hint" @click="inputText = hint"
                class="text-xs px-2 py-1 rounded bg-gray-100 dark:bg-gray-700 text-gray-500 hover:text-blue-500">
          {{ hint }}
        </button>
      </div>
    </div>
  </div>
</template>
```
